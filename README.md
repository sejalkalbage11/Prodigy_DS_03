## Task 3: Decision Tree Classifier

### Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service using the Bank Marketing dataset.

### Tools Used
- Python
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

### Steps Performed
- Loaded the dataset
- Encoded categorical variables
- Split the data into training and testing sets
- Trained a Decision Tree Classifier
- Evaluated the model
- Visualized the decision tree
- Analyzed feature importance

### Conclusion
The Decision Tree model was successfully trained and evaluated. The feature importance chart highlights the variables that most influenced the prediction of whether a customer subscribed to the term deposit.
